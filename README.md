# Amazon Global Sales Performance Dashboard

## Project Overview

This project is an interactive Power BI dashboard built to analyze Amazon's global sales performance from **2022–2024**. The dashboard helps stakeholders monitor revenue, profit, product performance, customer segments, and regional sales through interactive visualizations.

## Business Objectives

* Analyze overall sales performance
* Compare profit across product categories
* Identify top-performing products
* Compare customer segments
* Analyze sales by country
* Analyze regional product category sales
* Provide interactive filtering using slicers

## Insights

* Furniture generates the highest profit.
* Asia contributes the highest regional sales.
* India records the highest sales among all countries.
* Corporate customers place the highest number of orders.
* Tables and Laptops are the most profitable products.
